Instalar PostgreSQL

Abre una terminal y escribe

$ sudo apt update

Instala PostgreSQL

$ sudo apt install postgresql

Verifica la instalacion

$ sudo systemctl is-active postgresql
$ sudo systemctl is-enabled postgresql
$ sudo systemctl status postgresql

Checa que el servidor este listo para aceptar conexiones

sudo pg_isready

Crear la Base de Datos

Accede al usuario postgres

sudo su - postgres

Corre el comando psql

$ psql
postgres=#

Crea la base de datos

postgres=# CREATE USER tunombre WITH PASSWORD 'contraseña';
postgres=# CREATE DATABASE tunombredb;
postgres=# GRANT ALL PRIVILEGES ON DATABASE tunombredb TO tunombre;
postgres=# \q

Sal usando el comando exit

Instalar pgAdmin4

Este no esta disponible en un repositorio de Ubuntu como PostgreSQL por lo que hay que instalarlo de la siguiente manera, pero antes cambiaremos de usuario para que puedas copiar el comando mas facilmente

$ sudo su - root

Una vez que eres el usuario root ejecuta lo siguiente

$ curl https://www.pgadmin.org/static/packages_pgadmin_org.pub | apt-key add
$ sh -c 'echo "deb https://ftp.postgresql.org/pub/pgadmin/pgadmin4/apt/$(lsb_release -cs) pgadmin4 main" > /etc/apt/sources.list.d/pgadmin4.list && apt update'

Vuelve a tu usuario y ejecuta lo siguiente

$ sudo apt install pgadmin4

Configurando pgAdmin4

Abre pgAdmin4 te pedira que ingreses una contraseña Ahora haz click en Add New Server Te pedira que llenes las caracteristicas del nuevo servidor En Name le puse prueba Y en Host name se pone localhost asi como en Username pondras tu nombre de usuario y la contraseña A continuacion pulsamos Save y ya habremos configurado el servidor

Instalando una BD de Ejemplo

Descargaremos una base de datos como la del Curso Practico de SQL e iremos a Schemas de ahi a Create Schema Ahora llenaremos lo que nos pide Y listo ya tenemos nuestra BD, en el menu superior iremos a Query Tool Ahora iremos al icono de folder Y abriremos nuestro archivito Nos abrira la el editor de consultas Damos click en el boton de play y nos deberia retornar Query returned succesfully Finalmente comprobamos que todo esta correcto con un Query sencillo
